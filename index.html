本地写文章

选中你的Blog文件夹,右键单击，选择 Git Bash Here

新建文章

输入指令 hexo new xxx (xxx为文章标题)

写文章

编辑器写完，ctrl+s 保存

执行 hexo s 指令，浏览器打开 http://locahost:4000 即可看到预览效果，Ctrl+C 退出

渲染并部署网站

hexo g渲染生成网站

hexo d部署网站(把渲染好的html等网站文件，上传到GitHub服务器)
